## 強みの3本柱

**① 先手を打つリスク管理**
炎上PJ経験から「兆候の段階でリスク登録・対応策の準備」を習慣化。

**② バイリンガルPM**
発注者（ユーザー部門）と受注者（開発会社）の双方を経験。三者間の「翻訳者」として最適な着地点を導く。

**③ 業務起点のIT推進**
製造業15年・小売業経験を持つPMとして、「業務の目的」から要件を整理。現場が使えるシステムを実現。
